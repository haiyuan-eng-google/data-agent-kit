# Agent Evaluation

This directory contains recipes and quickstarts for measuring and evaluating the performance of your agents using frameworks like EvalBench.

## Core Initiatives

### EvalBench
[EvalBench](https://github.com/GoogleCloudPlatform/evalbench) is our recommended framework for structured agent evaluation. It allows you to:
- **Benchmark Performance:** Compare different model versions or prompt strategies.
- **Regression Testing:** Validate that new changes don't break existing capabilities.

#### Quickstarts
- [EvalBench Quickstart](https://colab.sandbox.google.com/github/GoogleCloudPlatform/evalbench/blob/main/docs/examples/sqlite_example.ipynb): Get started with your first evaluation suite.
